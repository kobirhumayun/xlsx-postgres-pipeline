# UI Pages (Suggested)

## /datasets
- list/create/edit datasets
- mapping editor must be usable (JSON form acceptable)

## /import
- upload Excel
- **Standard Mode**: select dataset + sheet
- **Flexible Mode**: select database + table + sheet
- run import
- show summary + errors

## /relationships
- define join mappings

## /reports
- choose report type
- set parameters
- run report
- show table
- export to Excel

## /query
- *New page*
- Sidebar: Schema Browser (DBs -> Tables)
- Main area: SQL Editor
- Actions: Run, Export to Excel
- Results: Data Grid
